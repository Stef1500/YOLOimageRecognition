# YOLOimageRecognition
A program that uses a custom trained yolov8 model to correctly recognize and locate blue 3d circles within a program (aimlabs).
Then calcualte the center of the generate square and move the mouse to the location and click it. It supports serveral circles at the same time.
model trained on a 5000 image dataset.

.py is named MANGUERASSIST as a tribute to a good friend who is not very good at videogames.
